Top Buchneuheiten Widget shows you the newest books every day

== Description ==
the Top Buchneuheiten Widget shows you the newest books every day in categories like political books, humans 2009, guidebooks, 	historical books, krimi, thriller, fiction  

Das Widget Top Buchneuheiten zeigt die neusten Bücher, die jeden Tag aktualisiert werden. Es werden Kategorien wie politische Sachbücher, Menschen 2009, Ratgeber, histrische Romane, Krimi, Thriller, Belletristik     


== Installation ==
English
 - Upload topbuchneuheiten to the '/wp-content/plugins/' directory
 - Activate the plugin through the 'Plugins' menu in WordPress
 - Klick on Design -> Widget and drag & drop "Top Buchneuheiten" into the active Widgets
 

Deutsch 
 - topbuchneuheiten ins '/wp-content/plugins/' hochladen
 - Aktiviere das Plugin
 - Wechsel zum Punkt Widget unter Design und ziehe das Plugin an die gewünschte Position
